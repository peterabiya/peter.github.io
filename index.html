01About

I am a cybersecurity engineer working across security operations, detection engineering, and AI security. He designs threat models before a line of code is written, builds the SIEM and IDS stack that watches the system afterward, and automates the response playbooks that fire when something's wrong.

My path into security wasn't linear. I started running daily operations for a computer training institute, moved into web development and instruction, then technical writing on cybersecurity topics for a general audience — before moving into hands-on security engineering. That range shows in how i work: as comfortable configuring a Docker Compose stack across four engineering teams writing an incident report a non-technical stakeholder can actually use.

My work at Expadox Lab as the Security Engineer, where i lead threat modeling for an AI-driven security platform and has built two production-style tools from the ground up: a detection platform (FireOps) and a SOAR automation layer (ShieldFlow).
